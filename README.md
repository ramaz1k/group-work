# Hotel Search Landing Page

Простой и современный дизайн сайта для поиска отелей. Макет создан в Figma и предназначен для удобного поиска, просмотра и бронирования гостиниц.

## О проекте

Данный лендинг помогает пользователям быстро находить подходящие отели, просматривать информацию о номерах и выбирать лучшие варианты для отдыха или путешествий.

## Секции сайта

### Header
Верхняя панель навигации с логотипом, меню и кнопками входа или регистрации.

### Hero Section
Главный экран с поисковой формой для выбора:
- города;
- даты;
- количества гостей;
- типа номера.

### Popular Hotels
Секция с популярными отелями и краткой информацией о них.

### Hotel Categories
Блок с категориями отелей:
- Luxury;
- Resort;
- Budget;
- Business;
- Family.

### Features Section
Информация о преимуществах сервиса:
- быстрый поиск;
- удобное бронирование;
- лучшие цены;
- поддержка клиентов.

### Testimonials
Отзывы пользователей о сервисе поиска и бронирования отелей.

### CTA Section
Призыв к действию для бронирования или начала поиска.

### Footer
Нижняя часть сайта с контактами, ссылками и социальными сетями.
