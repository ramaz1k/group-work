# Education Website Template

Современный образовательный сайт, разработанный в Figma. Дизайн предназначен для онлайн-обучения, презентации курсов и удобного взаимодействия студентов с образовательной платформой.

## О проекте

Этот проект представляет собой шаблон образовательного сайта, который помогает пользователям находить курсы, изучать новые навыки и получать качественное онлайн-образование.

## Секции сайта

### Hero Section
Главный экран с заголовком, описанием платформы и кнопкой для начала обучения.

### About Education
Информационный блок о возможностях платформы, преимуществах обучения и доступных программах.

### Courses Section
Секция с популярными курсами и направлениями обучения.

### Features
Блок с основными преимуществами платформы, включая удобный доступ к материалам, опытных преподавателей и гибкий график обучения.

### Instructors
Секция с информацией о преподавателях и экспертах.

### Student Reviews
Отзывы студентов о курсах и качестве обучения.

### Statistics
Блок со статистикой платформы, количеством студентов, курсов и преподавателей.

### CTA Section
Секция с призывом зарегистрироваться и начать обучение.

### Footer
Нижняя часть сайта с контактной информацией, навигацией и социальными сетями.

## Особенности дизайна

Макет выполнен в современном стиле с удобной структурой, чистой типографикой и адаптивным интерфейсом для разных устройств.

## Технологии

Для реализации проекта можно использовать HTML5, CSS3, JavaScript, React.js и Tailwind CSS.

## Responsive Design

Дизайн адаптирован для desktop, tablet и mobile устройств.

## Figma

Проект создан в Figma с использованием Auto Layout, компонентов и современных принципов UI/UX дизайна.
